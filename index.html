``` --- ### 🚀 1단계: 구글 파이어베이스(Firebase) 세팅 및 연동하기 앱 코드 안의 `firebaseConfig`를 내 것으로 채우기 위한 아주 간단한 과정입니다. 비용은 **완전 무료**입니다. 1. **파이어베이스 콘솔 접속:** [Firebase Console](https://console.firebase.google.com/)에 구글 계정으로 로그인하고 **[프로젝트 만들기]**를 누릅니다. (이름은 마음대로 작성) 2. **웹 앱 추가:** 프로젝트가 만들어지면 화면 중앙에 보이는 **`` (웹 마크)** 아이콘을 클릭해 앱을 등록합니다. 3. **Config 복사:** 등록이 완료되면 화면에 `const firebaseConfig = { apiKey: ... };` 모양의 설정 코드가 나옵니다. 이 6줄 정도의 코드를 복사해서 **위 HTML 코드 260번째 줄 근처의 `firebaseConfig` 부분에 그대로 덮어쓰기** 해줍니다. 4. **인증(Auth) 켜기:** 좌측 메뉴에서 **Authentication(인증) -> 시작하기 -> Sign-in method** 탭으로 갑니다. 목록 중에 **[익명]**을 찾아 '사용 설정'으로 켜고 저장합니다. (로그인 없이 누구나 닉네임만 치고 들어오게 하기 위함) 5. **데이터베이스(Firestore) 켜기:** 좌측 메뉴에서 **Firestore Database -> 데이터베이스 만들기**를 클릭합니다. 위치는 'asia-northeast3 (Seoul)'로 하고 생성합니다. 6. **규칙(Rules) 수정:** Firestore 메뉴 중 **[규칙]** 탭을 눌러서 아래 코드로 바꾸고 **[게시]**를 누릅니다. ```javascript rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if true; // 누구나 읽고 쓸 수 있게 허용 } } }
